# Mintstone

Comprehensive web admin panel for self-hosted Minecraft servers.

## Planned Features

- Authentication
- Server management
- Server monitoring
- Plugin and server updates
- Support for multiple servers
- Support for all server types (Vanilla, Spigot, Paper, etc.)
- ...and more!
